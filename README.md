## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Laniman/threejs-brain-animation.git
   ```
2. Install the dependencies:
   ```bash
   npm install
   ```
3. Run the app:
   ```bash
   npm run dev
   ```
